## 🚀 Sobre o Projeto
O **Elegant Choice** é uma vitrine digital de alta performance desenvolvida para o mercado de luxo. O foco principal é a **Experiência do Usuário (User Experience - UX)** e a conversão de vendas através de uma interface limpa e responsiva.

### 🛠️ Tecnologias e Arquitetura (Bolinhas e Caixas)
O projeto foi estruturado seguindo os padrões de mercado aprendidos no curso de ADS:

* **HTML5/CSS3**: Estrutura e estilização da "moldura de ouro".
* **Git/GitHub**: Controle de versão e deploy automatizado.
* **Ubuntu (Linux)**: Ambiente de desenvolvimento de alta performance.
* **Gestão de Ativos**: Organização rigorosa de imagens e dados de mercado.
